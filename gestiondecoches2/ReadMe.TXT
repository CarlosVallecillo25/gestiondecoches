Este programa en Java es una aplicación simple de gestión de coches que permite realizar diversas operaciones en un archivo de texto que almacena información sobre los coches. Aquí se proporciona una breve descripción de la estructura y funcionalidades del programa.

Estructura del Proyecto
El proyecto consta de dos paquetes principales:

negocio: Contiene la clase Coche, que representa un objeto Coche con atributos como marca, modelo, año y precio.

aplicacion: Contiene la clase Principal, que es la clase principal del programa con el método main como punto de entrada. Este paquete también incluye métodos para mostrar el contenido del archivo, agregar un coche, modificar un coche, eliminar un coche y otros métodos auxiliares.

Funcionalidades
El programa ofrece las siguientes funcionalidades:

Ver contenido del fichero: Muestra en la consola el contenido del archivo "fichero.txt" que almacena la información de los coches.

Agregar un coche: Permite al usuario agregar información sobre un nuevo coche, que se añade al final del archivo.

Modificar un coche: Permite al usuario modificar la información de un coche existente en el archivo.

Eliminar un coche: Permite al usuario eliminar la información de un coche existente en el archivo.

Salir: Termina la ejecución del programa.

Instrucciones de Uso

Al ejecutar el programa, se mostrará un menú con las opciones mencionadas.

El usuario puede seleccionar una opción ingresando el número correspondiente.

Para agregar un coche, el usuario debe proporcionar la marca, modelo, año y precio.

Para modificar un coche, el usuario debe ingresar la marca y modelo del coche que desea modificar, y luego proporcionar la nueva información.

Para eliminar un coche, el usuario debe ingresar la marca y modelo del coche que desea eliminar.

El contenido del archivo "fichero.txt" se muestra al seleccionar la opción de ver contenido.

La ejecución del programa puede pausarse entre operaciones.

Notas
El programa utiliza un archivo de texto para almacenar la información de los coches.
Se recomienda revisar y modificar el código según las necesidades específicas.
